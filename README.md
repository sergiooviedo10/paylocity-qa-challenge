# Paylocity QA Challenge

This repository contains my solution for the Paylocity QA Challenge.

## Scope
- Bug discovery and reporting
- API testing using Postman
- Automation execution artifacts
- Supporting screenshots and evidence

## Repository Structure

### bug-reports/
Contains documented defects categorized by severity.

### postman/
Contains Postman collections and environment configurations.

### automation/
Contains automation-related files and Newman execution artifacts.

### screenshots/
Contains screenshots supporting defect reports.

## Tools Used
- Postman
- GitHub
- REST APIs

## Notes
Sensitive credentials and confidential information have been excluded from this repository.
